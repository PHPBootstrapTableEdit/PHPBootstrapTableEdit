Quick Start Demo Using SQLite and PHP Webserver

    - PHP Requirements: PHP 7+ with mbstring and gd extensions enabled
    - Database Requirement: SQLite 3, or MariaDB/MySQL, or PostgresSQL

Step 1: Create Database

    SQLite:
        sqlite3 example.db < example.sql

    MariaDB/MySQl, PostgresSQL:
        see commented sections inside database.sql

Step 2: Start PHP's Webserver

    from inside the examples folder:
        php -S localhost:8080

Step 3: Open Web Browser to http://localhost:8080/

    0_basic.php
    basic usage

    1_typical.php
    typical usage - date formatting on index, validation hooks, image uploading

    2_dependent_selects.php
    related dropdown selects - when country changes, options in provinces change

    3_parent.php
    parent child relationship - when editing country, a link to manage provinces is displayed

    4_passwords.php
    password hashing, optional password input
